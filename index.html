Hoe doe je dit? Zie de instructies onderaan deze pagina.

Hoe maak je een JSONBin account?
1. Ga naar jsonbin.io → klik Sign Up
2. Maak een gratis account (alleen e-mail + wachtwoord)
3. Ga naar API Keys → kopieer je Master Key
4. Klik linksboven op + New Bin → plak {} → klik Create
5. Kopieer het Bin ID uit de URL (het lange stuk na /b/)
6. Vul beide velden hierboven in op elke telefoon/laptop die je gebruikt
